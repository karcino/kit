Kit — Orchestrator vs Narrator

Runtime vs Design-time. One substrate, two jobs.

Runtime agent
Orchestrator
does a thing in the world
Trigger
kit ask "book dinner at Luigi, 19:30"
Output
calendar event · route · travel buffer
Tools
kit.route · kit.cal · kit.docs · …
Latency
sub-second (fast-LLM, one-shot, parallel)
Fails when
wrong tool or wrong params
G2 role
does the demos
Design-time agent
Narrator
explains a thing about the system
Trigger
kit reflect "what if we add plugins?"
Output
diagrams · concept docs · scenarios
Tools
retrieval over code · notion · vault · git
Latency
quality > speed
Fails when
plausible-sounding nonsense
G2 role
documents the demos · is the demos
Narrator first · Orchestrator second · Both on the same Kit core
